nordea-visualization
====================

Project solid made for myself at the moment
Just to get some reality check on my current expenses

Reads a simple .csv file and visualizing it using d3.js

### Dependencies
        Node 0.10.x
        Mocha
        MongoDB

### Build
        
        $ npm install
        $ gulp

## Run tests
        $ npm install -g mocha
        $ cd test/models && mocha
